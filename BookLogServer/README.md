# BookLogServer

## Prerequisites
- Java 17
- Gradle 8.x

## Setup
1. Install Gradle
```bash
brew install gradle
```

2. Build the project
```bash
./gradlew build
```

3. Run the application
```bash
./gradlew bootRun
```

## Project Structure
- `src/main/java`: Java source code
- `src/main/resources`: Configuration files
- `build.gradle.kts`: Gradle build configuration
- `settings.gradle.kts`: Project settings
